# JAVA_SpringMVC-Hibernate_Customers-Manager

This Application uses SpringMVC, Hibernate and MySql.

It allows to:
- get all customers from the database,
- add a new customer,
- update a customer,
- get a single customer to pre-populate the customer-update form,
- delete a customer.
